# Planned Tools

Future tools and experiments are tracked as **Ixchel ideas** under
`.ixchel/ideas/` in this repository (git-first Markdown).

For active tools, see:

- [hbd](/docs/tools/hbd) — Issue tracker
- [ixchel](/docs/tools/ixchel) — Knowledge weaving system

<Callout type="info">
  These tools are in the planning stage. APIs and features may change.
</Callout>

## Contributing

Interested in helping build these tools? Check out the [GitHub repository](https://github.com/kevinmichaelchen/ixchel-tools) and look for issues labeled with the tool you're interested in.
